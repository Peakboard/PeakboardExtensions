# Peakboard Extension: Firebird

This extension provides a database connector for [Firebird](https://firebirdsql.org/) open-source SQL databases, allowing you to execute SQL queries directly from Peakboard.

## Custom List: Firebird Query

Execute SQL queries against a Firebird database and use the results as a Peakboard data source.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Connection String | Connection string to the Firebird database |
| SQL Statement | The SQL query to execute |

## Installation

1. Download `Firebird.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select the Firebird extension.
