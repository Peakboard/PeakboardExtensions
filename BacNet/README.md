# Peakboard Extension: BACnet

This extension provides BACnet protocol support for building automation systems. It allows you to discover BACnet devices on the network, browse their objects, and read property values.

## Custom Lists

### BacNet Devices
Discovers and lists all BACnet devices available on the network.

### BacNet Objects
Lists all objects of a specific BACnet device.

### BacNet Properties
Reads the property values of BACnet objects.

## Installation

1. Download `BacNet.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select one of the BACnet custom lists.
