# Peakboard Extension: Trello

This extension integrates Peakboard with [Trello](https://trello.com/), enabling you to display Trello boards, lists, and cards on your dashboards.

## Custom Lists

### Trello Boards
Lists all Trello boards accessible with the provided API key and token.

### Trello Cards
Retrieves cards from a specific Trello board, including card names, descriptions, labels, and list assignments.

## Configuration

| Parameter | Description |
|-----------|-------------|
| API Key | Trello API key (from https://trello.com/power-ups/admin) |
| Token | Trello authorization token |

## Installation

1. Download `Peakboard.Extensions.Trello.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select the desired Trello custom list.
