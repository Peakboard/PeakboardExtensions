# Peakboard Extension: Python

This extension allows you to execute Python scripts from Peakboard and use the output as a tabular data source. It is useful for running data transformations, calculations, or accessing systems through Python libraries.

## Custom List: Python Script

Executes a Python script and returns the output as a Peakboard data table.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Script | The Python script to execute |

## Prerequisites

Python must be installed on the Peakboard Box or the machine running the Peakboard Designer preview.

## Installation

This extension is currently available as source code only in the `SourceCodeNew` folder. Build the solution and package the output as a ZIP file to install it.
