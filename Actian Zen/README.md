# Peakboard Extension: Actian Zen

This extension provides a database connector for Actian Zen (formerly Pervasive SQL) databases, allowing you to run SQL queries directly from Peakboard.

## Custom List: Actian Zen Query

Execute SQL queries against an Actian Zen database and use the results as a Peakboard data source.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Connection String | ODBC connection string to the Actian Zen database |
| SQL Statement | The SQL query to execute |

## Installation

1. Download the ZIP file from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select the Actian Zen extension.

## Prerequisites

An Actian Zen ODBC driver must be installed on the Peakboard Box.
