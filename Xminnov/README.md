# Peakboard Extension: Xminnov RFID Reader

This extension integrates [Xminnov](https://www.xminnov.com/) RFID readers with Peakboard, allowing you to read RFID tag data and display it on your dashboards.

## Custom List: RFID Reader

Receives RFID tag data from a connected Xminnov reader.

## Installation

This extension is currently available as source code only in the `SourceCodeNew` folder. Build the solution and package the output as a ZIP file to install it.
