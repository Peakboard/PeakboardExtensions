# Peakboard Extension: Woutex e-Ink Displays

This extension integrates Peakboard with [Woutex](https://woutex.com/) e-Ink displays, enabling paperless digital signage in industrial and logistics environments.

## Custom List: Woutex Display

Controls and updates content on Woutex e-Ink displays from Peakboard.

## Documentation

For a step-by-step guide, see:

[ByeBye Paper - Going paperless with Peakboard, SAP, and Woutex e-Ink Displays](https://how-to-dismantle-a-peakboard-box.com/ByeBye-Paper-Going-paperless-with-Peakboard-and-Woutex-e-Ink-Displays.html)

## Installation

1. Download `Woutex.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **Woutex** from the extensions list.
