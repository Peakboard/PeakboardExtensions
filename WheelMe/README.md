# Peakboard Extension: wheel.me

This extension connects Peakboard to the [wheel.me](https://www.wheel.me/) autonomous robot fleet management platform. Monitor and control wheel.me robots directly from your Peakboard dashboard.

## Custom Lists

### WheelMe Robots
Lists all robots in the fleet with their current status and position.

### WheelMe Floors
Lists all floors/maps configured in the wheel.me system.

### WheelMe Positions
Lists all predefined positions (waypoints) where robots can navigate to.

## Installation

1. Download `WheelMe.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select the desired wheel.me custom list.
