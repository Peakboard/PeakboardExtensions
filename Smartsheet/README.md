# Peakboard Extension: Smartsheet

This extension integrates Peakboard with [Smartsheet](https://www.smartsheet.com/), allowing you to display project management data on your Peakboard dashboards.

## Custom Lists

### Smartsheet Table List
Lists all Smartsheet sheets accessible with the provided API token.

### Smartsheet Table Data
Retrieves the row and column data from a specific Smartsheet sheet.

## Configuration

| Parameter | Description |
|-----------|-------------|
| API Token | Smartsheet API access token |
| Sheet | The sheet to retrieve data from (selected from the list) |

## Installation

1. Download `Smartsheet.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select the desired Smartsheet custom list.
