# Peakboard Extension: AcmeData

This extension generates random sample data for demonstration and testing purposes. It is useful for prototyping Peakboard dashboards without connecting to a real data source.

## Custom List: AcmeData

Returns a table of randomly generated demo data. No configuration or connection parameters are required.

## Installation

1. Download `AcmeData.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **AcmeData** from the extensions list.
