# Peakboard Extension: WERMA Smart Monitor

This extension connects Peakboard to [WERMA](https://www.werma.com/) SmartMonitor signal tower monitoring systems, allowing you to display machine status and production signals on your Peakboard dashboards.

## Custom List: WERMA Data

Retrieves monitoring data from a WERMA SmartMonitor system, including signal tower states and machine status information.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Server | WERMA SmartMonitor server address |
| Credentials | Authentication details |

## Installation

1. Download `PeakboardExtensionWerma.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **WERMA** from the extensions list.
