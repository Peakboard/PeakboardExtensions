# Peakboard Extension: HubSpot

This extension integrates Peakboard with the [HubSpot](https://www.hubspot.com/) CRM platform, allowing you to query and display HubSpot data on your dashboards.

## Custom List: HubSpot Data

Retrieve data from HubSpot via the HubSpot API.

### Configuration

| Parameter | Description |
|-----------|-------------|
| API Key / Token | HubSpot API authentication token |
| Query | The API query to execute |

## Installation

1. Download `HubSpot.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **HubSpot** from the extensions list.
