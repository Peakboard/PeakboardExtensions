# Peakboard Extension: Network Files

This extension provides access to files on network shares from Peakboard. Use it to list and read files from UNC paths or mapped network drives.

## Custom List: Network Files

Lists files available on a network path.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Network Path | UNC path or mapped drive path to the network share |
| Credentials | Username and password for network share authentication |

## Installation

1. Download `NetworkFiles.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **Network Files** from the extensions list.
