# Peakboard Extension: Fix Resolution

This extension optimizes the Peakboard Runtime for 4K monitors by automatically scaling the display. Use it when your Peakboard dashboard appears too small on high-resolution screens.

## Custom List: Fix Resolution

Add this data source to your project to enable automatic display scaling. No configuration is required -- the extension detects the screen resolution and adjusts scaling accordingly.

## Installation

1. Download `FixResolution.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **Fix Resolution** from the extensions list.
