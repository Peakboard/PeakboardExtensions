# Peakboard Extension: MQTT Server

This extension turns the Peakboard Box into an MQTT server (broker), allowing other devices to publish and subscribe to MQTT messages directly through Peakboard.

## Custom List: MQTT Server

Hosts an MQTT broker on the Peakboard Box and provides access to received messages as a data source.

## Documentation

For a step-by-step guide, see:

[DIY Guide - Transform Your Peakboard Box into an MQTT Server](https://how-to-dismantle-a-peakboard-box.com/DIY-Guide-Transform-Your-Peakboard-Box-into-an-MQTT-Server.html)

## Installation

1. Download `MQTTServer.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **MQTT Server** from the extensions list.
