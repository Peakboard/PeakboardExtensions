# Peakboard Extension: Tableau Token Generator

This extension generates embedding tokens for [Tableau](https://www.tableau.com/) dashboards, enabling you to embed Tableau views into Peakboard applications with proper authentication.

## Custom List: Tableau Token Generator

Generates a Tableau trusted authentication token that can be used to embed Tableau views in a Peakboard web widget.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Tableau Server URL | URL of the Tableau Server or Tableau Online instance |
| Username | Tableau user account |
| Site | Tableau site name (leave empty for default site) |

## Installation

1. Download `Peakboard.Extensions.TableauTokenGenerator.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **Tableau Token Generator** from the extensions list.
