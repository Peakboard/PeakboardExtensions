# Peakboard Extension: Microsoft Dynamics 365

This extension connects Peakboard to the [Microsoft Dynamics 365](https://dynamics.microsoft.com/) API, providing access to ERP and CRM data from your Peakboard dashboards.

## Custom List: Dynamics 365 Data

Retrieve data from Microsoft Dynamics 365 entities via the Dynamics Web API.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Environment URL | Dynamics 365 environment URL |
| Tenant ID | Azure AD tenant ID |
| Client ID | Azure AD app registration client ID |
| Client Secret | Azure AD app registration client secret |
| Entity / Query | The Dynamics 365 entity or OData query |

## Documentation

[Unlocking Microsoft's Dataverse with Peakboard](https://how-to-dismantle-a-peakboard-box.com/Unlocking-Microsofts-Dataverse-with-Peakboard.html)

## Installation

1. Download `PeakboardExtensionMicrosoftDynamics365.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **Microsoft Dynamics 365** from the extensions list.
