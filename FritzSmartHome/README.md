# Peakboard Extension: AVM Fritz Smart Home

This extension connects Peakboard to [AVM Fritz](https://en.avm.de/) smart home devices, primarily thermostats. It allows you to monitor and control Fritz-based smart home devices from your Peakboard dashboard.

## Features

- Read thermostat temperature and settings
- Control thermostat target temperature
- Integrate Fritz smart home data into Peakboard dashboards

## Configuration

| Parameter | Description |
|-----------|-------------|
| Fritz!Box URL | URL of the Fritz!Box (default: http://fritz.box) |
| Username | Fritz!Box login username |
| Password | Fritz!Box login password |

## Installation

1. Download `FritzSmartHomeExtension.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **Fritz Smart Home** from the extensions list.
