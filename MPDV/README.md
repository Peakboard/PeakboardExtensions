# Peakboard Extension: MPDV

This extension connects Peakboard to [MPDV](https://www.mpdv.com/) Manufacturing Execution Systems (MES), enabling you to display production data and shop floor metrics on your Peakboard dashboards.

## Custom List: MPDV Data

Retrieve data from an MPDV MES system.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Server URL | URL of the MPDV server |
| Credentials | Authentication details for the MPDV API |
| Query | The data query to execute |

## Installation

1. Download `MPDV.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **MPDV** from the extensions list.
