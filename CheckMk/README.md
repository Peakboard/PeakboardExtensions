# Peakboard Extension: Check_MK

This extension connects Peakboard to the [Check_MK](https://checkmk.com/) IT monitoring system, allowing you to display monitoring data on your Peakboard dashboards.

## Custom List: CheckMk Data

Retrieves monitoring data from a Check_MK server instance.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Server URL | URL of the Check_MK server |
| Credentials | Authentication details for the Check_MK API |

## Installation

1. Download `CheckMk.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **CheckMk** from the extensions list.
