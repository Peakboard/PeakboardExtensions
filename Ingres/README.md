# Peakboard Extension: Ingres

This extension provides a database connector for [Ingres](https://www.actian.com/databases/ingres/) SQL databases, allowing you to execute SQL queries directly from Peakboard.

## Custom List: Ingres Query

Execute SQL queries against an Ingres database and use the results as a Peakboard data source.

### Configuration

| Parameter | Description |
|-----------|-------------|
| Connection String | Ingres connection string |
| SQL Statement | The SQL query to execute |

## Installation

1. Download `PeakboardExtensionIngres.zip` from the `Binaries` folder.
2. Add the extension to Peakboard Designer via **Manage Extensions**.
3. Add a new data source and select **Ingres** from the extensions list.
